## ファイル形式

### VRPファイル (.vrp)

VRPファイルは以下のセクションで構成される

#### ヘッダー情報

- `NAME`: インスタンス名
- `COMMENT`: 問題の説明（車両数、最適値など）
- `TYPE`: 問題タイプ（CVRP = Capacitated VRP）
- `DIMENSION`: ノード数
- `EDGE_WEIGHT_TYPE`: 距離計算方式
- `CAPACITY`: 車両容量

#### NODE_COORD_SECTION

ノードの座標情報（ノード番号、X座標、Y座標）

```
ノード番号 X座標 Y座標
1 0 0          # デポ（開始・終了地点）
2 3 5          # 顧客1
3 2.5 9        # 顧客2
...
```

#### DEMAND_SECTION

各ノードの需要量

```
ノード番号 需要量
1 0            # デポは需要量0
2 33           # 顧客1の需要量
3 15           # 顧客2の需要量
...
```

### 解ファイル (.sol)

最適解のルート情報

```
Route #1: ノード番号のリスト
Route #2: ノード番号のリスト
...
Cost: 総コスト
```

## 制約条件

1. **容量制約**: 各ルートの総需要量 ≤ 車両容量（2010）
2. **訪問制約**: 各顧客は必ず1回だけ訪問
3. **デポ制約**: 全ルートはデポから開始し、デポに戻る
4. **車両数制約**: 使用車両数 ≤ 4台
