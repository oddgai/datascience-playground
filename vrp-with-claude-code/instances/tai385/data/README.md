# TAI385 VRP Instance

## Instance Information
- **Name**: TAI385
- **Type**: Capacitated Vehicle Routing Problem (CVRP)
- **Dimension**: 386 nodes (1 depot + 385 customers)
- **Number of Vehicles**: 46
- **Capacity**: 65
- **Edge Weight Type**: EUC_2D (Euclidean distance in 2D)

## Instance Characteristics
This is a very large-scale CVRP instance from the TAI benchmark set, with 385 customers requiring service from a single depot. This is one of the most challenging instances in terms of scale.
